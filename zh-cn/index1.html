什么是Laragon？

Laragon 是一个可移植、隔离、快速且功能强大的通用开发环境，适用于 PHP、Node.js、Python、Java、Go、Ruby。它快速、轻便、易于使用且易于扩展。

Laragon 非常适合构建和管理现代 Web 应用程序。它专注于性能——围绕稳定性、简单性、灵活性和自由度而设计。

Laragon 非常轻巧，并且会尽可能保持精瘦。核心二进制文件本身小于 2MB，运行时使用的 RAM 小于 4MB。

Laragon 不使用 Windows 服务。它有自己的service orchestration异步和非阻塞管理服务，因此您会发现使用 Laragon 可以快速而流畅地运行。

享受！

特征

  • 漂亮的 URL
    使用app.test而不是localhost/app.
  • 便携式
    您可以毫无顾虑地移动 Laragon 文件夹（到另一个磁盘，另一个笔记本电脑，同步到云，...）。
  • 隔离的
    Laragon 与您的操作系统有一个隔离的环境 - 它会保持您的系统干净。
  • 操作简单
    不像其他人为你预先配置，Laragonauto-configs所有复杂的东西。这就是为什么您可以毫不费力地添加其他版本的 PHP、Python、Ruby、Java、Go、Apache、Nginx、MySQL、PostgreSQL、MongoDB 等。
  • 现代而强大的
    Laragon 配备了适合构建现代 Web 应用程序的现代架构师。您可以同时使用 Apache 和 Nginx，因为它们是完全托管的。
    此外，Laragon 使事情变得更容易：
    • 想要一个 Wordpress CMS？只需 1 次点击。
    • 想向客户展示您的本地项目吗？只需 1 次点击。
    • 想要启用/禁用 PHP 扩展？只需 1 次点击。
笔记

Laragon 是真正的隔离和便携。但是，您可能需要使用安装程序，因为它会检测并安装缺少的运行时组件，这些组件是运行使用 Visual Studio 构建的 C++ 应用程序（例如PHP, Apache为您）所需的。

多快？

Laragon 立即启动。您可以在 Windows 启动时设置 Laragon 自动运行。运行时，Laragon 只使用少量的 RAM。

有多容易？

  • Laragon 非常容易安装和升级：
    只需下载最新版本并点击Next, Next, Next...
  • Laragon 非常易于使用：
    几乎有一段时间，它可以开箱即用，您无需接触任何配置文件。
  • Laragon 非常容易扩展：
    您可以通过将其他服务解压缩到Laragon's bin文件夹来将其他服务添加到您当前的堆栈中。
